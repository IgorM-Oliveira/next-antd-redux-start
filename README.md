## Running locally

1. Clone this repo:

```sh
$ git clone https://github.com/IgorM-Oliveira/Portfolio.git
```

2. Then go to the project's folder:

```sh
cd Portfolio
```

3. Install all dependencies:

```sh
yarn install
```

4. Run locally:

```sh
yarn dev
```
